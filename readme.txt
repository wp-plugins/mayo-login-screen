Real time saver for designer and developer!== Description ==Tired of the pale looking, default Wordpress login screen? This plugin can help you to make your login screen stunning without writing a single line of code!With this plugin you can:<ol><li>Replace the Wordpress logo with your website name or your logo.</li><li>Change the color theme of the login form to match the color of your website.</li><li>Customize the background of the login screen. You can change the background color or upload an image to use as background.</li></ol>Once activated, you can go to your dashboard, Appearance > Login Screen to customize your Login screen.You can preview your login screen while you are customizing it!*This plugin does <b>NOT</b> create a new page for the login. Instead, it customize the Wordpress default login screen.<small><i>Credit to <a href="http://bgrins.github.io/TinyColor/">TinyColor</a> for color calculation in this plugin.</small>== Installation ==This section describes how to install the plugin and get it working.1. Goto WordPress Dashboard -> Add new plugin and search "mayo"2. Click Install button on "MAYO - Custom Login Screen"3. Activate the plugin through the 'Plugins' menu in WordPress4. Go to the plugins options page in the WordPress admin ('Themes' -> 'Login Screen')5. Play with the settings and save.Happy customizing your login screen!== Frequently Asked Questions ==== Screenshots ==1. Admin interface2. Built-in color picker3.
