You can change the background color or upload an image to use as background.</li></ol>Once activated, the setting page in dashboard located in Appearance > Login Screen.*This plugin does <b>NOT</b> create a new page for the login. Instead, it customize the Wordpress default login screen.<small><i>Credit to <a href="http://bgrins.github.io/TinyColor/">TinyColor</a> for color calculation in this plugin.</small>This plugin is brought to you by <a href="http://passionindesign.com/">PassionInDesign.com</a>== Installation ==This section describes how to install the plugin and get it working.1. Goto WordPress Dashboard -> Add new plugin and search "mayo"2. Click Install button on "MAYO - Custom Login Screen"3. Activate the plugin through the 'Plugins' menu in WordPress4. Go to the plugins options page in the WordPress admin ('Themes' -> 'Login Screen')5. Play with the settings and save.Happy customizing your login screen!== Frequently Asked Questions ==== Screenshots ==1.
