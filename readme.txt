Customize the default Wordpress login screen without getting your hands dirty with coding. Real time saver for designer and developer!
== Description ==
This plugin does <b>NOT</b> create a new page for the login. Instead, it allow you to customize the Wordpress default login screen without touching the code.
You can preview your login screen while you are customizing it!
== Installation ==
This section describes how to install the plugin and get it working.
1. Goto WordPress Dashboard -> Add new plugin and search "mayo"
2. Click Install button on "MAYO - Custom Login Screen"
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the plugins options page in the WordPress admin ('Themes' -> 'Login Screen')
5. Play with the settings and save.

Happy customizing your login screen!
